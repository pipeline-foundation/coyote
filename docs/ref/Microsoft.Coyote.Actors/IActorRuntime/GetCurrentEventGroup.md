# IActorRuntime.GetCurrentEventGroup method

Returns the current [`EventGroup`](../EventGroup.md) of the actor with the specified id. Returns null if the id is not set, or if the [`ActorId`](../ActorId.md) is not associated with this runtime. During testing, the runtime asserts that the specified actor is currently executing.

```csharp
public EventGroup GetCurrentEventGroup(ActorId currentActorId)
```

| parameter | description |
| --- | --- |
| currentActorId | The id of the currently executing actor. |

## Return Value

The current EventGroup or null.
